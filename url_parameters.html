Privly Test File

This file contains a sample of URL formats using new parameters recognized by the extensions. Privly extensions must pass all these test cases before deployment.

Anchor Elements

No Parameters

These are links without any URL parameters.

  • Localhost link
    http://localhost:3000/posts/2#privlyinject1

  • Privly link https://priv.ly/posts/2#privlyinject1

Forced Passive

These links are forced into passive mode by a parameter.

  • Localhost link
    http://localhost:3000/posts/2?passive=true#privlyinject1

  • Privly link https://priv.ly/posts/2?passive=true#privlyinject1

Passive Message

These links display a message when they are in passive mode.

  • Localhost link
    http://localhost:3000/posts/2?passiveMessage=You+are+in+passive+mode,+it+worked#privlyinject1

  • Privly link https://priv.ly/posts/2?passive=true&passiveMessage=You+are+in+passive+mode,+it+worked#privlyinject1

Forced Passive with Message

These links are forced into passive mode, but they have a message.

  • Localhost link
    http://localhost:3000/posts/2?passiveMessage=You+are+in+passive+mode,+it+worked&passive=true#privlyinject1

  • Privly link https://priv.ly/posts/2?passive=true&passiveMessage=You+are+in+passive+mode,+it+worked#privlyinject1

Forced Exclusion

These links are not modified once they are anchor elements.

  • Localhost link
    http://localhost:3000/posts/2?exclude=true#privlyinject1

  • Privly link https://priv.ly/posts/2?exclude=true#privlyinject1

Burnt

These links were tagged as being destroyed on the remote server.

  • Localhost link
    http://localhost:3000/posts/2?burntAfter=1#privlyinject1

  • Privly link https://priv.ly/posts/2?burntAfter=1#privlyinject1

Burnt with Message

These are links that are burnt, but they have a message that is viewable in place after the time is up.

  • Localhost link
    http://localhost:3000/posts/2?burntAfter=1&burntMessage=this+is+the+burn+message#privlyinject1

  • Privly link https://priv.ly/posts/2?burntAfter=1&burntMessage=this+is+the+burn+message#privlyinject1

Excluded by Attribute

These links are excluded not by parameters on the link, but by an attribute on the anchor element.

  • Localhost link
    http://localhost:3000/posts/2#privlyinject1

  • Privly link https://priv.ly/posts/2#privlyinject1
